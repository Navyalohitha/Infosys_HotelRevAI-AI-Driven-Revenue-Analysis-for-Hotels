# Infosys_HotelRevAI-AI-Driven-Revenue-Analysis-for-Hotels
**Project Overview**

The Infosys_HotelRevAI project focuses on analyzing hotel booking data to generate meaningful business insights using Microsoft Power BI. The project transforms raw booking and customer data into interactive dashboards that help hotel management monitor revenue performance, customer behavior, and operational efficiency.

This project demonstrates how business intelligence tools, data visualization, and analytical metrics can support decision-making in the hospitality industry.

**Objectives**

The main objectives of this project are:

Analyze hotel revenue and booking trends

Monitor operational performance such as occupancy and stay patterns

Understand customer booking behavior and cancellation trends

Build interactive dashboards for business insights

Forecast future revenue trends using data analytics

**Tools and Technologies**

Microsoft Power BI – Data visualization and dashboard development

DAX (Data Analysis Expressions) – Business metric calculations

Power Query – Data transformation and preparation

Excel / Dataset – Hotel booking data source

**Key Dashboards**
1. Hotel Revenue AI (Overview)

Provides a high-level overview of hotel performance.

Key Metrics:

Total Revenue

Total Bookings

Occupancy Rate

Cancellation Rate

Revenue distribution across hotel types

2. Executive Financial Overview

Focuses on financial performance and revenue trends.

Key Metrics:

Revenue Growth %

Average Daily Rate (ADR)

RevPAR (Revenue per Available Room)

Monthly revenue trends

3. Customer & Booking Behavior Analysis

Analyzes customer booking behavior and reliability.

Key Metrics:

Cancellation Rate

Repeat Guest Rate

Average Lead Time

Booking patterns

4. Operational & Stay Performance Dashboard

Evaluates hotel operational performance.

Key Metrics:

Weekday vs Weekend stays

Room utilization

Special guest requests

Parking space usage

5. Future Revenue & Target Performance Dashboard

Provides forecasting insights and performance benchmarks.

Key Metrics:

Revenue trend forecasting

Actual vs Target revenue comparison

Top earning hotel

Expected revenue for next 12 months

**Key Business Metrics**

The project uses several important hotel industry KPIs.

Total Revenue – Overall financial performance

Occupancy Rate – Percentage of rooms occupied

Average Daily Rate (ADR) – Average price paid per room

RevPAR – Revenue per available room

Cancellation Rate – Percentage of cancelled bookings

Average Lead Time – Average time between booking and arrival
