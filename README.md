**Hotel Revenue Analytics Dashboard**

My main project is Hotel Revenue Analytics Dashboard.

In this project, I worked on 50,000+ hotel booking records using Power BI.

First, I cleaned the data using Power Query.
Then I created important KPIs like:

Total Revenue
Occupancy Rate
ADR
RevPAR
Cancellation Rate

After that, I built interactive dashboards to analyze:

Revenue trends
Customer booking behavior
Cancellation patterns
Weekday vs weekend stays

I also created revenue forecasting to predict future performance.

This project helps hotel management understand business performance and make better decisions.
